# Package index

## Managing AirNow API tokens

An API token allows you to interact with the AirNow API. Tokens can be
created at [airnowapi.org](https://docs.airnowapi.org).

- [`get_airnow_token()`](https://briandconnelly.github.io/airnow/reference/get_airnow_token.md)
  [`set_airnow_token()`](https://briandconnelly.github.io/airnow/reference/get_airnow_token.md)
  : Manage AirNow API Tokens

## Retrieving air quality data

Retrieve forecasts and current/historical air quality data

- [`get_airnow_conditions()`](https://briandconnelly.github.io/airnow/reference/get_airnow_conditions.md)
  : Get current air quality conditions
- [`get_airnow_area()`](https://briandconnelly.github.io/airnow/reference/get_airnow_area.md)
  : Get air quality data for a given region
- [`get_airnow_forecast()`](https://briandconnelly.github.io/airnow/reference/get_airnow_forecast.md)
  : Get air quality forecast
